# EcoPath Healthcare Management System
Real-time healthcare resource management system with AI-powered disease outbreak monitoring and inventory optimization.

## Features
- **AI-Powered Nurse Reports**: Automatic extraction of disease, severity, and patient count using Gemini AI
- **Inventory Management**: Real-time stock tracking with anomaly detection
- **Weather Integration**: Correlate weather data with disease patterns
- **Analytics Dashboard**: Interactive visualizations with Plotly
- **Smart Data Aggregation**: Intelligent report consolidation and updates

## Architecture
- **Backend**: Spring Boot 3.x + Java 17
- **Database**: Snowflake Data Warehouse
- **AI**: Google Gemini AI
- **Frontend**: Streamlit
- **APIs**: OpenWeather API

## Prerequisites
- Java 17+
- Python 3.8+
- Maven 3.6+
- Snowflake Account
- Gemini API Key
- OpenWeather API Key
